# مراجعة وتحسين نظام الحصاد الملكي

## الغرض من البرومبت
مراجعة شاملة لنظام الحصاد الملكي في تطبيق Royal Door لتحويله إلى نظام عالمي متكامل يغطي جميع جوانب الحصاد من شراء الباقات إلى سحب الأرباح وتحويل الأموال.

## المهام الرئيسية للمراجعة

### 1. تحليل النظام الحالي
- فحص صفحات الحصاد الموجودة (`harvest_stats_page.dart`, `investment_page.dart`)
- مراجعة نماذج البيانات (`user_model.dart`) للرصيد والمحافظ
- تحليل آليات الشراء والسحب والتحويل

### 2. تحديد النقص في النظام
- **شراء الباقات**: هل تغطي جميع طرق الدفع الدولية؟
- **إدارة الرصيد**: هل تدعم العملات المتعددة؟
- **الجواهر والعملات**: هل هناك نظام تحويل شامل؟
- **سحب الأرباح**: هل تدعم الحسابات البنكية الدولية؟
- **تحويل الأموال**: هل تدعم التحويلات الدولية الآمنة؟

### 3. اقتراح التحسينات المطلوبة
- إضافة دعم للعملات الدولية (USD, EUR, GBP)
- تكامل مع بوابات الدفع الدولية (Stripe, PayPal)
- نظام تحويل العملات التلقائي
- إضافة خيارات السحب البنكي الدولي
- تحسين أمان التحويلات بين المستخدمين
- إضافة سوق عالمي لتداول الباقات

### 4. التنفيذ العملي
- إنشاء صفحات جديدة للميزات المفقودة
- تحديث قاعدة البيانات لدعم العملات الجديدة
- إضافة خدمات الدفع والتحويل
- تحسين واجهات المستخدم للدعم الدولي

## المتطلبات التقنية
- Firebase للتخزين والمعالجة
- Flutter للواجهات
- خدمات الدفع الخارجية
- نظام أمان متقدم للمعاملات المالية

## الخطوات المقترحة للتنفيذ
1. تحليل الكود الحالي وتحديد النقص
2. تصميم النظام المحسن
3. تطوير الميزات الجديدة
4. اختبار شامل للنظام
5. النشر التدريجي
