# Royal Admin Panel

لوحة تحكم ويب حديثة لإدارة بوت رويال دور.

## الميزات الأساسية:
- إحصائيات المستخدمين
- إدارة المكافآت
- إدارة التحديات
- إدارة التذاكر
- إرسال إشعارات
- دعم لغات متعددة
- صلاحيات للمسؤولين فقط
- تصميم متجاوب وجذاب
- جميع المقترحات الملكية

## التقنيات المستخدمة:
- React
- Firebase

## خطوات التشغيل:
1. تثبيت الحزم: `npm install`
2. بدء التطوير: `npm start`
